# Md Nasir Khan — Portfolio

Personal portfolio site of **Md Nasir Khan**, a Full Stack Engineer with 3+ years
of experience building production backend systems and web applications, with a
strong focus on the **Java & Spring Boot** ecosystem.

🔗 **Live:** https://mnkkhan77.github.io/

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tooling)
- **Tailwind CSS** (styling)
- **lucide-react** (icons)
- **GitHub Actions** → **GitHub Pages** (CI/CD)

## Sections

- **Hero** — intro, stats, and résumé download
- **About** — background and core strengths
- **Experience** — work history (Growth Loops, Cognizant) + education & training
- **Skills** — primary, current, cloud/DevOps, and other competencies
- **Projects** — featured full-stack projects with live and source links
- **Contact** — email, phone, GitHub, and LinkedIn

## Getting Started

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview

# Lint
npm run lint
```

## Deployment

Every push to `main` triggers the
[`Deploy Vite site to GitHub Pages`](.github/workflows/deploy.yml) workflow,
which builds the site and publishes `dist/` to GitHub Pages.

> Pages **Source** must be set to **GitHub Actions**
> (Settings → Pages → Build and deployment).
